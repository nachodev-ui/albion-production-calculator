## Alcance

Este sitio documenta el estado vigente de `albion-craft-calculator`. Las notas de
entregas antiguas se consolidaron en un archivo histórico y dejaron de ocupar la
raíz del repositorio.

La documentación normativa se divide en:

- **Arquitectura:** responsabilidades, límites y flujos entre módulos.
- **Funcionalidades:** comportamiento visible y reglas de negocio.
- **Operación:** instalación, configuración, pruebas y mantenimiento.
- **Decisiones:** ADR breves para elecciones que afectan a varios módulos.
